## Workflow

1. Edit the `.md` file(s).
2. Tell Cursor which file/section changed, e.g. *“Update the site from `content/homepage.md` hero section.”*
3. Cursor updates the React components under `src/components/` and `src/pages/`.

The live site keeps copy in those component files. Image paths, layout, and styling stay in code.
