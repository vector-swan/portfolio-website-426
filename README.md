# Mia Donnell Portfolio

Personal portfolio site for Mia Donnell, Senior Product Designer at Workday.

## Stack

Vite + React + TypeScript + Tailwind + shadcn/ui.

## Local development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

## Notes

The site is password protected. Password lives in `src/components/PasswordGate.tsx` and is included with portfolio submissions when applying to roles.
